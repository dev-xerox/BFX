# 🤖 BFX

> **Мощный ИИ-агент** нового поколения с интерфейсом в стиле [Kiro](https://kiro.dev)

---

## ⚡ Возможности

| Функция | Описание |
|---------|----------|
| 🔍 **Веб-Поиск** | Мгновенный поиск информации через DuckDuckGo |
| 💻 **Терминал** | Выполнение команд и скриптов в реальном времени |
| 🌐 **Веб-Фетч** | Загрузка и парсинг веб-страниц |
| 🕷️ **Firecrawl API** | Глубокий скрейпинг и индексация через Firecrawl |

## 🧠 Поддерживаемые AI-провайдеры

- **ChatGPT** (OpenAI) — `gpt-4o` и другие
- **Claude** (Anthropic) — `claude-sonnet-4` и другие
- **Gemini** (Google) — `gemini-2.5-pro` и другие
- **Grok** (xAI) — `grok-3` и другие

---

## 🚀 Установка

```bash
# Клонируем репозиторий
git clone https://github.com/dev-xerox/BFX.git
cd BFX

# Устанавливаем зависимости
pip install -r requirements.txt

# Копируем конфиг и добавляем API-ключи
cp .env.example .env
# Отредактируйте .env и раскомментируйте нужный провайдер
```

## 🎮 Запуск

```bash
python bfx.py
```

## 📖 Команды

| Команда | Описание |
|---------|----------|
| `/help` | Показать справку |
| `/clear` | Очистить историю чата |
| `/provider <name>` | Сменить провайдера (`chatgpt`/`claude`/`gemini`/`grok`) |
| `/model <name>` | Сменить модель |
| `/search <query>` | Веб-поиск |
| `/exec <command>` | Выполнить команду в терминале |
| `/fetch <url>` | Загрузить веб-страницу |
| `/firecrawl <url>` | Firecrawl скрейпинг |
| `/config` | Показать текущую конфигурацию |
| `/quit` | Выйти |
